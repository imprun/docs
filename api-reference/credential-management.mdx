---

외부 서비스(위하고, 홈택스 등)의 로그인 자격증명을 Vault에 안전하게 저장하고 관리합니다.
저장된 자격증명은 [스크래핑 워크플로우](/api-reference/workflows)에서 자동 로그인에 활용됩니다.

## 인증

모든 엔드포인트는 [API 키 인증](/authentication)이 필요합니다.

```bash
X-API-Key: {keyId}.{secret}
```

## 지원 프로바이더

| 프로바이더 | 값 | 설명 |
|-----------|------|------|
| 위하고 | `wehago` | 더존비즈온 위하고 |
| 홈택스 | `hometax` | 국세청 홈택스 |

---

## 자격증명 생성

<ParamField path="POST" type="/v2/credentials" />

새 자격증명을 등록합니다.

### 요청 본문

<ParamField body="provider" type="string" required>
  프로바이더 (`wehago`, `hometax`)
</ParamField>
<ParamField body="portal_id" type="string" required>
  로그인 ID
</ParamField>
<ParamField body="password" type="string" required>
  로그인 비밀번호 (저장 후 조회 불가)
</ParamField>
<ParamField body="label" type="string">
  식별용 라벨 (예: "본사 경리팀")
</ParamField>

<CodeGroup>
```bash cURL
curl -X POST https://prod.fn.imprun.dev/v2/credentials \
  -H "X-API-Key: {keyId}.{secret}" \
  -H "Content-Type: application/json" \
  -d '{
    "provider": "wehago",
    "label": "본사 경리팀",
    "portal_id": "user@wehago.com",
    "password": "password123"
  }'
```

```bash imprun-cli
imprun-cli credential register \
  --provider wehago \
  --portal-id user@wehago.com \
  --label "본사 경리팀"
# 비밀번호는 프롬프트로 입력
```
</CodeGroup>

### 응답 — `201 Created`

```json
{
  "id": "cred-49fb8ddd-9d38-49fb-85dd-90176517640e",
  "provider": "wehago",
  "label": "본사 경리팀",
  "portal_id": "user@wehago.com",
  "created_at": "2026-03-09T05:37:41Z"
}
```

<ResponseField name="id" type="string">자격증명 ID (`cred-` 접두사 + UUID)</ResponseField>
<ResponseField name="provider" type="string">프로바이더</ResponseField>
<ResponseField name="label" type="string">식별용 라벨</ResponseField>
<ResponseField name="portal_id" type="string">로그인 ID</ResponseField>
<ResponseField name="created_at" type="string">생성 시각 (RFC3339)</ResponseField>

## 자격증명 목록 조회

<ParamField path="GET" type="/v2/credentials" />

등록된 자격증명 목록을 조회합니다.

### 쿼리 파라미터

<ParamField query="provider" type="string">
  특정 프로바이더만 필터링 (생략 시 전체)
</ParamField>

<CodeGroup>
```bash cURL
curl https://prod.fn.imprun.dev/v2/credentials \
  -H "X-API-Key: {keyId}.{secret}"
```

```bash imprun-cli
imprun-cli credential list
imprun-cli credential list --provider wehago
```
</CodeGroup>

### 응답 — `200 OK`

```json
{
  "credentials": [
    {
      "id": "cred-49fb8ddd-9d38-49fb-85dd-90176517640e",
      "provider": "wehago",
      "label": "본사 경리팀",
      "portal_id": "user@wehago.com",
      "created_at": "2026-03-09T05:37:41Z"
    }
  ]
}
```

## 자격증명 수정

<ParamField path="PUT" type="/v2/credentials/{id}" />

라벨 또는 비밀번호를 변경합니다. 전달된 필드만 업데이트됩니다.

### 요청 본문

<ParamField body="label" type="string">변경할 라벨</ParamField>
<ParamField body="password" type="string">변경할 비밀번호</ParamField>

### 쿼리 파라미터

<ParamField query="provider" type="string" default="wehago">
  프로바이더
</ParamField>

```bash cURL
curl -X PUT https://prod.fn.imprun.dev/v2/credentials/cred-49fb8ddd-... \
  -H "X-API-Key: {keyId}.{secret}" \
  -H "Content-Type: application/json" \
  -d '{"label": "본사 총무팀", "password": "newpassword456"}'
```

### 응답 — `200 OK`

수정된 자격증명 객체를 반환합니다.

## 자격증명 삭제

<ParamField path="DELETE" type="/v2/credentials/{id}" />

### 쿼리 파라미터

<ParamField query="provider" type="string" default="wehago">
  프로바이더
</ParamField>

<CodeGroup>
```bash cURL
curl -X DELETE https://prod.fn.imprun.dev/v2/credentials/cred-49fb8ddd-... \
  -H "X-API-Key: {keyId}.{secret}"
```

```bash imprun-cli
imprun-cli credential delete cred-49fb8ddd-...
```
</CodeGroup>

### 응답 — `204 No Content`

<Warning>
  삭제된 자격증명을 사용하는 워크플로우는 `CREDENTIAL_NOT_FOUND` 에러로 실패합니다.
  삭제 전 연관된 워크플로우가 없는지 확인하세요.
</Warning>

## 오류 코드

| HTTP | 코드 | 설명 |
|------|------|------|
| 400 | `INVALID_REQUEST` | 필수 필드 누락 또는 잘못된 요청 |
| 400 | `INVALID_PROVIDER` | 지원하지 않는 프로바이더 |
| 401 | `UNAUTHORIZED` | API 키 누락 또는 인증 실패 |
| 404 | `CREDENTIAL_NOT_FOUND` | 자격증명을 찾을 수 없음 |
| 500 | `INTERNAL_ERROR` | 서버 내부 오류 |

## 보안

<Note>
  자격증명은 [HashiCorp Vault](https://www.vaultproject.io/)에 암호화되어 저장됩니다.
  비밀번호는 저장 후 어떤 API로도 조회할 수 없습니다.
</Note>

- **암호화 저장**: Vault KV v2 시크릿 엔진 사용
- **비밀번호 미노출**: 조회 응답에 비밀번호가 포함되지 않습니다
- **계정 격리**: 자격증명은 API 키의 계정 단위로 격리됩니다
- **TLS 전송**: 모든 통신은 HTTPS로 암호화됩니다

## 요금
