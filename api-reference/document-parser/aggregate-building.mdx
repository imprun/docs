집합건물 등기부등본은 `property_type`이 `"aggregate_building"`인 문서입니다. 아파트, 오피스텔, 상가 등 구분소유 건물에 해당하며, **전유부분**과 **대지권** 정보를 포함합니다.

## 표제부 (title_info)

집합건물의 표제부는 1동 건물 정보, 전유부분, 대지권 토지, 대지권 비율의 4개 섹션으로 구성됩니다.

### 집합건물 전용 필드

| 필드 | 타입 | 설명 |
|------|------|------|
| `building_name` | `string \| null` | 건물명 (예: `"게토333지식산업센터 제나동"`) |
| `exclusive_area` | `float \| null` | 전용면적 (㎡) |
| `land_right_ratio` | `string \| null` | 대지권 비율 (예: `"25455.4분의 67.06"`) |
| `land_right_area` | `float \| null` | 대지 전체 면적 (㎡) |
| `exclusive_part_entries` | `array` | 전유부분 항목 |
| `land_right_entries` | `array` | 대지권 토지 항목 |
| `land_right_ratio_entries` | `array` | 대지권 비율 항목 |

### 건물과 공유하는 필드

| 필드 | 타입 | 설명 |
|------|------|------|
| `road_address` | `string \| null` | 도로명주소 |
| `structure` | `string \| null` | 건물 구조 |
| `roof_type` | `string \| null` | 지붕 종류 |
| `floors` | `integer` | 지상 층수 |
| `building_type` | `string \| null` | 건물 용도 |
| `total_floor_area` | `float` | 1동 연면적 (㎡) |
| `areas` | `array` | 1동 층별 면적 |
| `building_entries` | `array` | 1동 건물의 표시 항목 |

---

## 전유부분 (ExclusivePartEntry)

해당 호실의 건물 정보입니다.

| 필드 | 타입 | 설명 |
|------|------|------|
| `display_number` | `string` | 표시번호 |
| `receipt_date` | `string` | 접수일자 |
| `building_number` | `string` | 동·호수 (예: `"제15층 제1501호"`) |
| `building_detail` | `string` | 건물내역 (구조·면적) |
| `cause_and_other` | `string` | 등기원인 및 기타사항 |
| `is_cancelled` | `boolean` | 말소 여부 |

---

## 대지권 토지 (LandRightEntry)

집합건물이 위치한 대지 토지 정보입니다.

| 필드 | 타입 | 설명 |
|------|------|------|
| `display_number` | `string` | 표시번호 |
| `location` | `string` | 소재지번 |
| `land_type` | `string` | 지목 (예: `"대"`) |
| `area` | `string` | 면적 (예: `"25455.4㎡"`) |
| `cause_and_other` | `string` | 등기원인 및 기타사항 |

---

## 대지권 비율 (LandRightRatioEntry)

해당 호실의 대지 지분 비율입니다.

| 필드 | 타입 | 설명 |
|------|------|------|
| `display_number` | `string` | 표시번호 |
| `land_right_type` | `string` | 대지권 종류 (예: `"1 소유권대지권"`) |
| `land_right_ratio` | `string` | 대지권 비율 (예: `"25455.4분의 67.06"`) |
| `cause_and_other` | `string` | 등기원인 및 기타사항 |
| `is_cancelled` | `boolean` | 말소 여부 |

<Note>
  대지권이 있는 집합건물의 경우, 을구의 근저당권에 별도의 공동담보목록이 없을 수 있습니다.
  대지권은 전유부분의 처분에 따르므로 별도 등기가 불필요합니다.
</Note>

---

## 갑구·을구

집합건물 등기부등본의 갑구·을구는 [공통 스키마](/api-reference/document-parser#갑구-section_a)와 동일합니다. 집합건물 특유의 등기 유형은 다음과 같습니다:

- **신탁**: 을구에 신탁 원부 번호가 기재됨
- **가처분**: 분양권 관련 처분금지 가처분
- **근저당권부채권압류**: 을구 근저당권에 대한 채권 압류
