## 크레딧 기반 요금제

imprun.dev는 **선불 크레딧** 기반 요금제입니다. 월 구독이나 약정 없이 사용한 만큼만 지불합니다.

### 특징

- **충전 단위**: ₩ 원
- **최소 충전**: ₩1,000
- **만료 없음**: 충전한 크레딧은 만료되지 않습니다
- **실시간 차감**: API 호출 시 즉시 차감
- **잔액 확인**: 콘솔에서 실시간 확인 가능

## API별 단가

| API | 엔드포인트 | 단가 |
|-----|-----------|------|
| 등기부등본 PDF 파서 | `POST /v1/document-parser/registry` | 콘솔에서 확인 |

<Info>
  API별 최신 단가는 [콘솔](https://imprun.dev/console/account)에서 확인하세요.
</Info>

## 과금 방식

1. API 호출 시 사용량이 기록됩니다
2. 5분 주기로 사용량을 집계하여 크레딧을 차감합니다
3. 잔액이 부족하면 API 호출이 거부됩니다 (HTTP 403)

## 충전

[콘솔의 크레딧 충전](https://imprun.dev/console/account) 메뉴에서 토스페이먼츠를 통해 충전할 수 있습니다.
