## 1. 회원가입

[imprun.dev/signup](https://imprun.dev/signup)에서 계정을 만드세요.

## 2. API 키 발급

콘솔의 **API Keys** 메뉴에서 새 키를 생성합니다.

<Warning>
  API 키는 생성 시 한 번만 표시됩니다. 안전한 곳에 저장해 주세요.
</Warning>

키 형식: `{keyId}.{secret}` (예: `a1b2c3d4...abcd1234...`)

## 3. 크레딧 충전

콘솔의 **크레딧 충전** 메뉴에서 원하는 금액을 충전하세요. 최소 ₩1,000부터 가능합니다.

<Info>
  회원가입 시 테스트용 보너스 크레딧이 지급됩니다.
</Info>

## 4. 첫 API 호출

등기부등본 PDF 파일을 준비하고 아래 명령을 실행하세요:

<CodeGroup>
```bash cURL
curl -X POST https://prod.fn.imprun.dev/v1/document-parser/registry \
  -H "Authorization: Bearer YOUR_KEY_ID.YOUR_SECRET" \
  -F "file=@등기부등본.pdf"
```

```python Python
import requests

url = "https://prod.fn.imprun.dev/v1/document-parser/registry"
headers = {"Authorization": "Bearer YOUR_KEY_ID.YOUR_SECRET"}
files = {"file": open("등기부등본.pdf", "rb")}

response = requests.post(url, headers=headers, files=files)
print(response.json())
```

```javascript Node.js
const fs = require('fs');

const form = new FormData();
form.append('file', new Blob([fs.readFileSync('등기부등본.pdf')]));

const response = await fetch('https://prod.fn.imprun.dev/v1/document-parser/registry', {
  method: 'POST',
  headers: { 'Authorization': 'Bearer YOUR_KEY_ID.YOUR_SECRET' },
  body: form,
});

const data = await response.json();
console.log(data);
```
</CodeGroup>

응답으로 구조화된 JSON 데이터를 받게 됩니다. 자세한 응답 스키마는 [API Reference](/api-reference/document-parser)를 참고하세요.
